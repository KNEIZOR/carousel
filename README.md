## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Слайдер передвигается мышкой, а также самостоятельно через 3 секунды, на декстоп отображается 4 слайдера, на мобильном 1 слайдер.
